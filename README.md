# causal-llm-research

本项目致力于从底层设计上革新大语言模型，赋予其内生的因果洞察与推理能力。其核心目标是研发出其推理过程本身即为因果推理的新型大语言模型。本项目由在该领域拥有深厚研究背景的龚鹤扬博士主导，并将包含其在该方向上的诸多理论研究成果。
